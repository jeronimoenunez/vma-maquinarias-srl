# VMA Máquinas — Landing (Astro)

High-performance landing page for VMA Máquinas, designed with a performance-first, scalable, and secure frontend architecture.
This project serves both as a production website and as a portfolio-grade reference for modern frontend architecture using Astro.

## Tech Stack
- Astro (static-first)
- React Islands (interactive filtering)
- TypeScript (strict)
- Domain-driven UI contracts

## Architecture highlights
- UI decoupled from data and domain logic
- Section registry pattern for scalable page composition
- Typed UI contracts (props as domain interfaces)
- Static-first rendering with optional client hydration
- Prepared for API / ML / AI integrations
- Security headers baseline (OWASP-oriented)

## Key Features
- Modular, reusable section system
- Interactive equipment filtering (React Island)
- Scroll-based UI effects
- Performance-optimized asset loading
- SEO-friendly static output
- Clean separation between:
    - UI
    - Domain
    - Data
    - Behavior

## Security Considerations
- Designed with OWASP Top 10 awareness
- Strict TypeScript typing
- No unnecessary client-side JavaScript
- Ready for:
    - CSP headers
    - Secure API routes
    - Auth / RBAC extensions
